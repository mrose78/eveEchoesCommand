GOOGLE SHEETS SETUP
1. Create a Google Sheet with these tabs:
• Mining — Date, System, Ore, Amount, ISK, Duration, Notes
• Ships — Name, Class, Status, Price, Role, Notes, Faction
• Upgrades — Name, Category, Cost, Priority, Desc, Done
• SafeZones — System, Region, Security, Safety, Ores, Notes
2. Get a Google Sheets API key:
console.cloud.google.com → APIs & Services → Sheets API → Credentials → Create API Key
3. Share your sheet (Anyone with link → Editor)
4. Get your Sheet ID from the URL:
docs.google.com/spreadsheets/d/THIS_PART/edit
5. Paste both above and click CONNECT
All data is also saved locally in your browser as a backup.
