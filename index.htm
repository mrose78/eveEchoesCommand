GOOGLE SHEETS SETUP
Your sheet needs these tabs (exact names):
• Character — col A = Field Name, col B = Value (key-value pairs, see below)
• Mining — cols A-G: Date, System, Ore, Amount, ISK, Duration, Notes
• Ships — cols A-I: Name, Class, Status, Price, Role, Omega, Tech, Faction, Notes
• SafeZones — cols A-F: System, Region, Security, Safety, Ores, Notes
Character tab — add these rows (A=field name, B=your value):
Character Name · Corporation · Alliance
Account Type (Alpha or Omega) · Omega Expires
Avatar URL (direct image link) · Home Station · Home Region
ISK Balance · ISK Updated · PLEX Balance
Skill Points · Skill Points Updated
Training Skill · Training ETA
Skill Queue (semicolon-separated: Mining IV;Shield IV)
Current Ship · Current Ship Class
Kills · Losses · Security Status
Standings (semicolon-separated: Caldari State:5.0;Gallente:-2.1)
ISK Goals (semicolon-separated: Retriever:50000000:12000000)
Log Date · Captain's Log
Row 1 in each tab = header row (will be skipped)
Share settings:
Sheet must be set to Anyone with the link → Viewer for the API key to read it
Status values for Ships tab (col C):
Owned · Wanted · Not Owned · Sold
Omega column (col F):
TRUE or FALSE (or YES / NO)
This dashboard is read-only. Edit data directly in your Google Sheet, then click ⟳ LOAD FROM SHEET to refresh.
